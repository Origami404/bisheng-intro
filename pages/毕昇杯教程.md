- 总览
	- 做什么
		- SysY -> ARMv7
		- 源语言特性:
			- int/float 两种类型
			- 加减乘除模运算表达式
			- 分离的比较表达式
				- 比较只能在控制结构的条件里用, 不能嵌入到普通表达式中
				- 要做短路
			- 表达式里对 int 和 float 采取相同的 op, 需要做合理的类型推断和隐式转换的插入
			- if 和 while 控制结构
			- 函数
				- 会调用外部函数
					- 外部函数作为静态链接库被链接
					- 基本上是 IO 函数
				- 不会被外部函数调用
			- 数组
				- 数组会作为函数 (包括一些外部函数) 的参数被传递, 此时会退化为指针
				- 语言中没有指针的概念
				- 有全局数组和局部数据 (栈上的)
